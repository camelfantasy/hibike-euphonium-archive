Hibike! Euphonium Archive project

Welcome to the website interface of the Hibike! Euphonium Archive project. Read the information below to get started.
Quick start

Use the search bar or click on a tag to find all images containing your query. To see all images, click on the All Images tab.

All tags can be found in the Tags tab, and are also listed on each image's page. Untagged images can be found here.

The folder structure can be navigated from each image and folder page, with the Root Folder tab linked to the topmost folder.
Permissions

All users with accounts have permission to add or remove existing tags from individual images, as well as all images in a folder at once (will not affect subfolders).

Administrators additionally have the ability to add or remove tags, create or delete user accounts, and sync data from Google Drive to the local database. Tags' categories can be changed by adding a tag as usual but selecting a different category. Database access is currently required for directly renaming tags.

The Root user is the sole person able to add or remove administrator accounts.
