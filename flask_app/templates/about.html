About

The Hibike! Euphonium Archive project was created by members of the Liz and the Blue Bird discord server, with the goal of creating a comprehensive archive of all physical and digital visual content that have been created for the Hibike! franchise. All files are hosted on Google Drive; this website is simply for search purposes.

This website is created and maintained by camelfantasy.
Contact: Discord @camelfantasy#6654 | Twitter: @camelfantasy

All content belong to their respective owners.

Changelog

Any notable changes will be listed here. This site is currently in beta.

0.6.0
Added image favorite functionality and page for all users with accounts.
Allowed anyone to create an account and delete it.
Staff can modify and delete accounts ranked below them.
Changed tag deletion to a dropdown list, changed tag editing to its own functionality, and moved all three tag functions into a single tabbed pane.
Tag addition, modification, or removal on the tag page will re-sort all the tags.
Tags added to images on image pages will be displayed in sorted position.
Added confirmation popup if adding a new tag to an image or folder.
Replaced all remaining unnecessary page reloads with ajax.
Removed username requirement for authentication on restricted API routes.

0.5.2
Changed Google Drive sync function to operate asynchronously.

0.5.1
Added full boolean expression functionality to advanced search.

0.5.0
